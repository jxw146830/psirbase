psiRbase: A database containing putative siRNA sequences (psiRNAs) from developmental data sets.
This website is a resource for scientists interested in the endo-siRNA population in small RNA datasets.
Searches will return any psiRNA/mRNA reverse complementary pairs with up to two mismatches, if desired.
To query, select the species, choose one of the two search options, and type the name of the mRNA or a putative endo-siRNA sequence.

miRNA, piRNA, and endogenous-small interfering RNAs (endo-siRNAs) are approximately 18-32 nucleotides long and play an important role in gene regulation. While miRNA and piRNA are highly conserved across species and well annotated, endo-siRNAs are challenging to identify because they are not well conserved. Small RNA libraries across various developmental stages have been collected and aligned to genomes. Mapped reads have been further processed to remove known rRNA, tRNA, miRNA, and piRNA sequences. The remaining putative endo-siRNAs have been annotated, given unique identifiers, and characterized by developmental stage. Currently, only wild-type small RNA libraries are included.

siRBase At present, SiRBase contains endo-siRNA sequences from four Species:
  • Caenorhabdits elegans(ce11)
  • Drosophila melanogaster(dm6)
  • Danio rerio(danRer10)
  • Mus Musculus(mm10)
More species will be added soon, please stay tuned!!!!

Fasta sequences from acquired datasets were first aligned to the indicated genome build using bowtie -v 1 -a --best --strata parameters to get exact genome coordinates. Any fafsta sequences that also mapped to a custom small RNA reference including annotated rRNA, tRNA, miRNA, and piRNA were excluded from the results Custom RNA reference made using rRNA sequences from https://genome.ucsc.edu/cgi-bin/hgTables, miRNA sequences from http://www.targetscan.org/vert_71/, piRNA sequences from http://regulatoryrna.org/database/piRNA/download.html, and tRNA sequences from http://gtrnadb.ucsc.edu/.
