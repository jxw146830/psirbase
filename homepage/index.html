psiRbase: putative siRNA (psiRNA)
A database and resource for scientists interested in endogenous-small interfering RNA sequences (endo-siRNAs).

Available developmental data sets have been collected through literature searches and Gene accession omnibus and removed of annotated small RNAs.

The search tool below will return putative endo-siRNA/mRNA reverse complementary pairs with up to two mismatches, if desired.
To query, select the species, choose one of the two search options, and type a putative endo-siRNA sequence or the name of the mRNA.

Background
miRNA, piRNA, and endo-siRNAs are generally 18-32 nucleotide sequences that play an important role in gene regulation.
While miRNA and piRNA are highly conserved across species and well annotated, endo-siRNAs are challenging to identify because they are not well conserved.
About psiRbase
To identify putative endo-siRNAs, sequences and small RNA libraries were collected and rigorously processed using the following procedure.
Sequences were aligned to genomes using bowtie allowing one mismatch (-v 1 -a --best --strata parameters).
Sequences were then aligned to a custom RNA reference to exclude currently annotated rRNA, tRNA, miRNA and piRNA sequences.
Information for the custom RNA references were found using the following resources:
•    UCSC

•    GrTNAdb

•    Targetscan

•    miRBase

•    piRbase

•    microRNA

•    mirDB

•    miRTar

•    PITA Catalog

•    piRNA Bank
Unmappable RNA reference sequences were then aligned to transcriptomes to identify exclusively antisense-siRNAs using same bowtie parameters.
Only sequences with four or fewer alignments were considered "putative endo-siRNA".
